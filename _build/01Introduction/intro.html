Introduction

Welcome to an interactive Jupyter Book for the research paper by Daniel B. Larremore entitled "TBayes-optimal estimation of overlap between populations of fixed size".

This Jupyter Book lets you interact with a demo of their code. You can change the code by inline on the pages or launch a MyBinder session to run your code in a Jupyter Notebook in another tab. The figures are made interactive using Plotly.
